# CSV Утилита на Python3

Консольная утилита для работы с CSV файлами на Python 3.

## Установка и запуск

1. Рекомендуемая версия python: 3.14.
2. Скачайте или клонируйте репозиторий на свой компьютер.
3. Перейдите в директорию проекта:
   ```bash
   cd путь/к/проекту
   ```
4. Создайте виртуальную среду и установите зависимости:
	```bash
	python -m venv env
	. env/bin/activate на Linux или env\Scripts\activate на Windows
	pip install -r requirements.txt
	```
5. Запустите тесты чтобы проверить что всё работает:
	```bash
	pytest -v
	```
6. Запустите и посмотрите инструкцию:
	```bash
	python -m main -h
	```
Пример запуска также на изображении launch_example.png
